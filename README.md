1. Install Tesseract-OCR:
   Windows: https://digi.bib.uni-mannheim.de/tesseract/
   Linux: sudo apt install tesseract-ocr
		    sudo apt install libtesseract-dev
2. pip install pytesseract

# Text-to-speech modules:
